# IELTS Learning

This workspace is organized around the four IELTS skills:

- `listening/`
- `reading/`
- `writing/`
- `speaking/`

Each skill has folders for notes, practice, mistakes, and resources. Use `AGENTS.md` as the repo guide for future Codex work.

## Weekly Use

1. Add lesson notes to the relevant `notes/` folder.
2. Save timed practice attempts in `practice/` with a dated filename.
3. Move repeated problems into `mistakes/` so they are easy to review.
4. Keep useful links, prompts, vocabulary, and source references in `resources/`.
