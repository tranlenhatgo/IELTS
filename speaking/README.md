# Speaking

Use this folder for IELTS Speaking Part 1, Part 2, and Part 3 notes, answer practice, pronunciation notes, and feedback.

## Folder Use

- `notes/`: answer frameworks, fluency strategies, topic vocabulary, and pronunciation notes.
- `practice/`: dated speaking drills, cue-card answers, recordings notes, and mock-test logs.
- `mistakes/`: repeated grammar, pronunciation, fluency, and vocabulary problems.
- `resources/`: question banks, topic lists, useful phrases, and feedback references.

## Suggested Focus

- Part 1: direct answers with one extra detail
- Part 2: cue-card structure and story control
- Part 3: opinion, comparison, cause, effect, and speculation
- Fluency: fewer long pauses and fewer memorized-sounding answers
