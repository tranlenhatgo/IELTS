# Speaking Part 2

Skill: Speaking  
Part: Part 2 cue card  
Source: user notes + provided screenshots

## Core Constraint

- Preparation time: 1 minute
- Speaking time: 1 to 2 minutes
- During prep: write abbreviations, not full sentences

## Skeleton

Use two layers:

| Layer | Purpose | Output |
|---|---|---|
| Horizontal brainstorm | Create main angles for the topic. | 3 to 5 aspects |
| Vertical brainstorm | Extend each angle into a story/detail. | examples, memory, reason, result |

## Adjective Rule

One adjective = one main idea.

Do not list adjectives like this:

`kind, generous, friendly, hardworking`

Use one adjective and attach a precise explanation:

`generous -> always willing to help those in need`

Band 8 value comes from the explanatory phrase, not from listing many simple adjectives.

## Horizontal Brainstorm

List describable aspects of the main object/person/place/event.

Example cue card: `Describe a person you admire`

| Aspect | Abbreviation Examples |
|---|---|
| Appearance | tall, neat, smile |
| Personality | generous -> help those in need |
| Talents | articulate -> explain complex ideas simply |
| Achievements | built support network, helped poor kids |

Purpose: gives main ideas so the answer does not run out after 30 seconds.

## Vertical Brainstorm

Turn each main idea into a small story.

| Aspect | Story Extension |
|---|---|
| Personality | generous -> donated money, food, and clothes to poor children in remote areas |
| Result | helped 300 kids; built 10 communal houses |
| Opinion | proud of her kindness; others looked up to her |

Purpose: makes the answer longer, more natural, and more convincing.

## What / Why / How / Result / Opinion

Example cue card: `Describe a person you admire` -> `Mom`

| What | Why | How | Result | Opinion |
|---|---|---|---|---|
| Personality | generous -> always willing to help those in need | donated money, food, and clothes to poor children in remote areas | helped 300 kids; built 10 communal houses | I am proud of her kindness; others looked up to her |

## Specific Vocabulary Rule

Avoid generic words when a specific phrase is available.

| Weak | Stronger |
|---|---|
| kind | generous; compassionate |
| help people | help those in need |
| poor people | poor children in remote areas |
| give things | donate money, food, and clothes |
| people respect her | people looked up to her |

## Answer Flow

1. Direct answer: identify the person/place/event/object.
2. Context: how you know it or when it happened.
3. Choose one strong adjective or quality.
4. Add the explanatory phrase.
5. Give one specific example.
6. Add result and opinion.

## Prep Note Style

Do not write sentences. Use compressed notes.

Bad prep:

`She is kind and generous because she helped me when I had a problem at school.`

Better prep:

`generous -> help those in need -> poor kids remote areas -> donate food/clothes -> 300 kids -> proud`

## Band 8 Focus

- Use notes to control direction, not memorize a script.
- One adjective becomes one developed idea.
- Use specific vocabulary, not generic safe words.
- Build a story from evidence, not a list of traits.
- Extend with cause, result, contrast, and personal reaction.
- Keep speaking even if one bullet is weak; develop the strongest angle.

## Quick Rule

Horizontal = ideas.  
Vertical = story.
Adjective + explanation + example + result + opinion = developed answer.
