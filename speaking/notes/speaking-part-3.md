# Speaking Part 3

Skill: Speaking
Part: Part 3 discussion
Source: user notes + screenshot

## Role

- Part length: about 4 to 5 minutes.
- Answer length: usually 4 to 6 sentences.
- Part 3 extends the topic from Part 2.
- It is a two-way discussion: the examiner can ask follow-up questions based on what you said.
- Difficulty rises gradually so the examiner can test the limit of your language.
- This part helps decide whether the answer can move into a higher band.

## Main Shift

Part 2 can be personal.
Part 3 should be more objective.

| Weak Part 3 Direction | Stronger Part 3 Direction |
|---|---|
| `I like...`; `In my life...`; `When I was...` | `young people...`; `working adults...`; `parents...`; `local communities...`; `society...` |
| personal story | group-level reason |
| one example from yourself | social trend, common behavior, wider effect |

Only focus on yourself if the question directly asks about you. This is rare in Part 3.

## Scope Ladder

Move upward when answering:

`myself -> people around me -> age group / job group -> society -> government / schools / companies -> future trend`

Band 8 focus: stay around group / society / future trend, not personal memory.

## Answer Shape

| Move | Function |
|---|---|
| Direct position | Answer the question clearly. |
| Group focus | Talk about people in society, not yourself. |
| Main idea | Give one clear argument. |
| Explanation | Explain why this happens or what it causes. |
| Example | Prove the point with a social example, not a personal story. |
| Balance | Add contrast if the question allows it. |
| Stop cleanly | Leave space for the examiner to challenge or follow up. |

Core logic:

`main idea -> explanation -> example`

## One Answer Structure

Use this when the question is broad enough for a developed answer.

| Step | Job | Note |
|---|---|---|
| What | Give a general answer to the question. | 1 sentence, direct and objective. |
| Main idea 1 | First argument. | Pick one clear social reason/effect. |
| Example 1 | Proof for idea 1. | Use a group/society example. |
| Main idea 2 | Second argument. | Add contrast, another group, or another effect. |
| Example 2 | Proof for idea 2. | Keep it short; do not turn it into a story. |

Pattern:

`general answer -> reason 1 -> example 1 -> reason 2 -> example 2`

## Useful Frames

- `For many young people, the main issue is...`
- `In bigger cities, this is becoming more common because...`
- `From a social point of view, the benefit is...`
- `The downside is that...`
- `This probably depends on age, income, and education level.`
- `In the long term, I think this could lead to...`

## Common Mistakes

- Turning Part 3 into another Part 2 story.
- Talking too much about personal experience.
- Giving a long monologue instead of a discussion-style answer.
- Making a claim without reason, effect, or contrast.
- Using only simple groups like `people`; be more specific.
- Answering too short, so the examiner has no language to assess.

## Quick Rule

Part 3 = Part 2 topic, but lifted to society level.
Personal answer only if the question directly asks about you.
