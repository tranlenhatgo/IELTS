# Reading

Use this folder for IELTS Reading strategies, passage practice, vocabulary from texts, and mistake tracking.

## Folder Use

- `notes/`: question-type strategies, skimming/scanning notes, and passage workflow.
- `practice/`: dated practice logs for passages, sections, and full tests.
- `mistakes/`: repeated errors such as false/not given confusion, keyword traps, and time loss.
- `resources/`: article links, vocabulary lists, practice-test references, and useful explanations.

## Suggested Focus

- True, False, Not Given
- Matching headings
- Matching information
- Multiple choice
- Sentence completion
- Time management
