# IELTS Reading Question Types Review Map

Skill: Reading  
Test type: IELTS General Training  
Source: user-provided image; use as a review checklist, not a guaranteed prediction

## Passage Map

| Passage | Question Types | Main Risk |
|---|---|---|
| Passage 1 | Sentence completion; True / False / Not Given | Accuracy under easy-text pressure |
| Passage 2 | Matching information; Multiple choice | Paraphrase and distractors |
| Passage 3 | Matching features; Matching headings | Main idea vs detail |

## Question-Type Notes

| Question Type | 8.0 Focus | Common Trap |
|---|---|---|
| Sentence completion | Predict grammar before searching; copy exact words; respect word limit. | Correct idea but wrong word form, too many words, or grammar mismatch. |
| True / False / Not Given | Prove exact agreement, contradiction, or absence from the text only. | Treating missing information as `False`; ignoring qualifiers like `only`, `most`, `may`, `always`. |
| Matching information | Match the function or meaning of the sentence, not surface keywords. | Paragraph contains same topic words but not the requested information. |
| Multiple choice | Eliminate partial truths; choose the option fully supported by context. | Option is true somewhere in the text but does not answer the question. |
| Matching features | Track who did, said, believed, found, or caused what. | Matching by nearest name instead of the actual claim/action. |
| Matching headings | Identify paragraph purpose, not an example or repeated keyword. | Heading matches one detail but not the whole paragraph. |

## Fast Review Order

1. True / False / Not Given
2. Matching headings
3. Matching information
4. Multiple choice
5. Sentence completion
6. Matching features

## Band 8 Rule

Do not hunt words. Hunt meaning, function, and limits.
