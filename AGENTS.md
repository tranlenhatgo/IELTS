# IELTS Learning Repo Instructions

## Purpose

This repo is for IELTS General Training review notes.

Current learning context:

- Test type: IELTS General Training.
- Target level: Band 8.0.
- The user is not using this repo for a test countdown.
- Primary job: turn user-provided IELTS information into concise Markdown notes for later review.

## Folder Map

- `listening/`: listening structure, parts, question types, traps, and review notes.
- `reading/`: reading structure, sections/passages, question types, traps, and review notes.
- `writing/`: Task 1 letters, Task 2 essays, formats, structures, traps, and review notes.
- `speaking/`: Part 1, Part 2, Part 3, answer structures, traps, and review notes.
- `study-plan/`: repo context and learning-focus notes only, unless the user explicitly asks for a plan.

Each skill folder uses the same structure:

- `notes/`: stable lesson notes, strategy notes, grammar notes, and useful patterns.
- `practice/`: dated practice attempts and timed-session logs.
- `mistakes/`: repeated errors, corrected examples, and review checklists.
- `resources/`: links, book references, prompt lists, vocabulary lists, and source material notes.

## Working Rules For Codex

- Prefer creating or updating repo-local Markdown files over chat-only advice.
- Do not overwrite existing notes unless explicitly asked. Prefer appending a clearly labeled section or editing the relevant stable topic file.
- Assume the user is aiming for Band 8.0 unless told otherwise.
- Keep notes concise, non-beginner, and focused on structure, traps, execution, and review value.
- Do not write beginner definitions for obvious IELTS terms unless the user asks.
- Do not add "How to use this note" sections unless they contain real study value.
- Prefer dense tables, short checklists, and high-signal bullets over long explanations.
- Use simple English unless the user asks for academic detail.
- Use dated filenames only for practice logs: `YYYY-MM-DD-topic.md`.
- Use stable topic filenames for notes, such as `reading-question-types-review-map.md` or `task-1-letter-structure.md`.
- Do not create a separate dated note for every image or source. Merge the information into the relevant stable topic note.
- Keep each note focused on one exam part, question type, answer format, grammar point, or strategy area.
- When reviewing writing or speaking answers, separate the original answer, corrected answer, feedback, score estimate, and next action.
- For General Training Writing Task 1, treat letters as a priority topic. Always identify purpose, tone, bullet coverage, opening, closing, and paragraphing.
- For General Training Reading, prioritize sections 1 and 2 speed plus section 3 accuracy under time pressure.
- Do not delete or reorganize user notes unless explicitly asked.
- Do not paste large copyrighted IELTS passages or book content. Store a source reference or short excerpt instead.
- If asked about current test rules, fees, booking, score policy, or official band descriptors, verify with official IELTS sources before answering.
