# Writing

Use this folder for IELTS Writing Task 1 and Task 2 notes, essay plans, drafts, feedback, and rewrites.

## Folder Use

- `notes/`: task structures, grammar patterns, linking language, and band-descriptor notes.
- `practice/`: dated essay plans, timed drafts, rewrites, and review logs.
- `mistakes/`: repeated grammar, coherence, task response, and vocabulary problems.
- `resources/`: question banks, model-answer references, vocabulary lists, and official descriptors.

## Suggested Focus

- Task 1 overview and trend language
- Task 2 position and paragraph structure
- Cohesion without overusing linking words
- Complex sentences with control
- Error correction and rewrite cycles
