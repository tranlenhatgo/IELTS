# Task 2 Agree / Disagree: 100% Position

Skill: Writing
Part: Task 2 argumentative essay
Question type: `To what extent do you agree or disagree?`

## Position

Use this route when you fully agree or fully disagree with the statement.

Core rule: every body paragraph must support the same side.

## Structure

| Paragraph | Job |
|---|---|
| Introduction | Paraphrase topic + give full agreement/disagreement. |
| Body 1 | Reason 1 + explanation + example/result. |
| Body 2 | Reason 2 + explanation + example/result. |
| Body 3 | Optional reason 3 if needed for word count/depth. |
| Conclusion | Restate full position + summarize reasons. |

## Fixed Frames

Introduction:

`Many people argue that [topic]. I completely agree/disagree with this view because [reason 1], [reason 2], and [reason 3].`

Body:

- `The main reason why I support/reject this view is that...`
- `Another reason is that...`
- `Finally, this view is convincing/problematic because...`

Conclusion:

`In conclusion, I completely agree/disagree that [topic]. This is mainly because [reason 1], [reason 2], and [reason 3].`

## Band 8 Focus

- Do not discuss the opposite side unless it directly strengthens your argument.
- Use one clear reason per body paragraph.
- If using only 2 bodies, each body must be developed deeply.
- If ideas are simple, use 3 bodies to reach 250+ words safely.

## Quick Review

Full position = one side only.
Body paragraphs = reasons for that side.
