# Task 1 Process Diagrams

Skill: Writing
Part: Task 1 process diagram
Source: forecast process questions

## Structure

| Paragraph | Job | Focus |
|---|---|---|
| Introduction | Paraphrase the process. | `how [item] is produced / generated / processed` |
| Overview | State the start, end, and main transformation. | no numbers unless shown |
| Body 1 | Early stages. | extraction, preparation, transport |
| Body 2 | Later stages. | conversion, final product, by-products |

## Overview Formula

`Overall, the process begins with [first stage] and ends with [final output], after [main transformation].`

Coal-to-electricity example:

`Overall, coal is extracted and processed before being converted into gas, which is then cleaned and used to generate electricity. The process also produces waste heat.`

## Body Split

| Process Type | Body 1 | Body 2 |
|---|---|---|
| Manufacturing | raw material + preparation | processing + final product |
| Energy production | fuel source + conversion | power generation + distribution |
| Natural process | early natural stages | later natural stages / cycle |
| Recycling loop | collection + sorting | treatment + reuse |

## Useful Language

| Function | Patterns |
|---|---|
| Sequence | `first`; `then`; `after this`; `subsequently`; `finally` |
| Passive process | `is mined`; `is crushed`; `is transported`; `is converted into`; `is cleaned`; `is sent to` |
| Transformation | `is turned into`; `is used to produce`; `generates`; `results in` |
| By-product | `waste heat is produced`; `gas is released`; `the remaining material is removed` |
| Loop | `is returned to`; `is recycled back into`; `the cycle then repeats` |

## Band 8 Focus

- Follow the arrows, not the visual size.
- Use passive voice when the actor is not important.
- Mention by-products or feedback loops if shown.
- Do not invent reasons, benefits, or problems.
- Do not use trend language unless the diagram contains time/data.

## Quick Review

Intro = what process.
Overview = first stage + final output + main transformation.
Bodies = early stages, then later stages.
