# Task 2 Essays

Skill: Writing
Part: Task 2 essay
Source: core IELTS structure note

## Target

- Minimum: 250 words.
- Practical target: 270 to 290 words.
- Time: about 40 minutes.
- Structure: usually 4 paragraphs.

## Core Structure

| Paragraph | Job | Must Include |
|---|---|---|
| Introduction | Rephrase the topic and give position. | clear stance / essay direction |
| Body 1 | First main idea. | topic sentence, explanation, example, result |
| Body 2 | Second main idea. | topic sentence, explanation, example, result |
| Conclusion | Final answer. | restated position, no new idea |

## Body Paragraph Formula

`Main idea -> explain the logic -> specific example -> result / link back to question`

Band 8 body paragraphs are not longer because they list more ideas. They are stronger because one idea is developed properly.

## Question Type Control

| Question Type | Essay Direction |
|---|---|
| `To what extent do you agree or disagree?` | Pick a clear position. Body 1 and Body 2 should both support that position, unless using a controlled partial-agree structure. |
| `Discuss both views and give your opinion.` | Body 1 = view A. Body 2 = view B + your opinion, or opinion made clear in intro and conclusion. |
| `Do the advantages outweigh the disadvantages?` | Compare weight, not number. One strong advantage can outweigh several weak disadvantages. |
| `What are the problems and solutions?` | Body 1 = problems/causes. Body 2 = solutions. Match each solution to a real problem. |
| `Do you think this is a positive or negative development?` | Treat like opinion. Choose positive, negative, or mixed, then defend it consistently. |
| Two-part question | Body 1 answers question 1. Body 2 answers question 2. Do not merge both answers randomly. |

## Introduction

Good intro = topic paraphrase + position.

Pattern:

`While [opposite side / background], I believe that [clear position] because [reason 1] and [reason 2].`

Avoid:

- memorized generic openings
- giving examples in the introduction
- unclear position like `This essay will discuss both sides` when the question asks for your opinion

## Band 8 Focus

- Answer the exact command in the question.
- Keep the same position from introduction to conclusion.
- Use one central idea per body paragraph.
- Explain cause and effect clearly.
- Use specific examples, not vague claims.
- Use precise topic vocabulary, not forced "academic" words.
- Make contrast controlled: `However`, `This does not mean`, `Despite this`.
- Conclusion must not introduce new arguments.

## Useful Patterns

| Function | Patterns |
|---|---|
| Main reason | `This is mainly because...`; `The strongest reason is that...` |
| Effect | `As a result...`; `This can lead to...`; `This makes it harder for...` |
| Contrast | `However...`; `This does not necessarily mean that...`; `Despite this...` |
| Example | `For example...`; `A clear example of this is...` |
| Judgement | `Therefore, I believe...`; `For this reason, ... is more convincing.` |

## Common Traps

- Writing balanced discussion when the task asks for a clear opinion.
- Listing 3 to 4 weak ideas instead of developing 2 strong ones.
- Using personal stories as the main evidence.
- Repeating the same idea in both body paragraphs.
- Forgetting to answer the second part of a two-part question.
- Making the conclusion stronger or different from the introduction.

## Quick Review

Plan the position first.
Two body paragraphs = two developed ideas.
Examples prove the logic; they do not replace the logic.
