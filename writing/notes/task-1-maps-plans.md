# Task 1 Maps / Plans

Skill: Writing
Part: Task 1 map / plan comparison
Source: user-provided map / plan questions

## Two Question Types

| Type | What It Shows | Main Strategy |
|---|---|---|
| Change over time | same place before/after | group by changed area/function |
| Feature comparison | two layouts/rooms/places at the same time | group by shared and different features |

## Intro Paraphrase

Goal: identify the visual, the place/layout, and whether the task shows change or comparison.

Change-over-time formula:

`The plans compare the layout of [place] in [year 1] and after changes in [year 2].`

Feature-comparison formula:

`The plans compare the layout and facilities of [item A] and [item B].`

Examples:

- `The plans compare the layout of a museum in 1998 and after its redevelopment in 2008.`
- `The plans compare the layout and facilities of student rooms designed for one and two people.`

| Question Word | Safer Paraphrase |
|---|---|
| picture / plans | the plans / the diagrams |
| show | compare / illustrate |
| plan of | layout of |
| after some changes | after its redevelopment / following several changes |
| student room for two people | a two-person student room |

Rules:

- Do not describe changes in the introduction.
- Use change language only if there are two time points.
- Use comparison language if both layouts are current alternatives.

## Structure: Change Over Time

| Paragraph | Job | Focus |
|---|---|---|
| Introduction | Paraphrase the question. | `The plans compare...` |
| Overview | 2 to 3 biggest changes. | main redevelopment, unchanged areas |
| Body 1 | Group by layout / function. | exhibition rooms, central space, entrance |
| Body 2 | Group by facilities. | cafe, restaurant, WC, shop, bags/coats |

Do not write one full paragraph for the old map and one full paragraph for the new map. Compare by category.

## Structure: Feature Comparison

Use this when the task compares two layouts, such as a room for two people vs a room for one person.

| Paragraph | Job | Focus |
|---|---|---|
| Introduction | Paraphrase the two items being compared. | `The plans compare the layout and facilities of...` |
| Overview | State the biggest differences and main similarities. | size, capacity, shared facilities |
| Body 1 | Compare size and main living/study areas. | dimensions, beds, desks, windows |
| Body 2 | Compare facilities and equipment. | bathroom, kitchen area, cupboard, table/chairs |

Do not use redevelopment language. There is no time change.

## Overview Formula

Change over time:

`Overall, the [place] was redesigned to [main purpose], with [major change], while [unchanged feature] remained largely unchanged.`

Museum example:

`Overall, the museum was redesigned to provide more visitor facilities, while the central garden was removed and the permanent exhibition rooms on the right side remained largely unchanged.`

Feature comparison:

`Overall, the [larger / first layout] is [main difference], whereas the [smaller / second layout] is [main contrast] but still includes [shared core features].`

Student room example:

`Overall, the two-person room is larger and contains duplicated sleeping and study areas, whereas the single room is more compact but still includes the main living facilities.`

## Grouping Method

Change over time:

| Group | What To Compare |
|---|---|
| Exhibition / main function | permanent rooms, temporary room, garden, central area |
| Visitor facilities | restaurant, cafe, seating area, WC, shop, bags/coats |
| Movement / access | entrance, ticket area, corridors, position changes |
| Unchanged features | rooms or areas that stayed in the same place |

Feature comparison:

| Group | What To Compare |
|---|---|
| Size / capacity | dimensions, number of people, amount of space |
| Main areas | beds, desks, living/study space |
| Shared facilities | bathroom, sink, stove, microwave, cupboard |
| Extra / missing features | table and chairs, bookshelf, TV, windows |

## Useful Verbs

Change over time:

| Change Type | Verbs |
|---|---|
| New feature | `was added`; `was introduced`; `was built` |
| Removal | `was removed`; `was demolished`; `disappeared` |
| Replacement | `was replaced by`; `gave way to`; `was converted into` |
| Movement | `was relocated to`; `was moved to`; `was shifted to` |
| Size change | `was expanded`; `was reduced`; `was enlarged` |
| No change | `remained unchanged`; `continued to be used as`; `was still located` |

Feature comparison:

| Function | Language |
|---|---|
| Similarity | `both rooms contain`; `each room has`; `the two layouts include` |
| Contrast | `whereas`; `while`; `in contrast`; `compared with` |
| Quantity | `twice as many`; `one fewer`; `two separate`; `a single` |
| Size | `larger`; `narrower`; `more spacious`; `more compact` |
| Missing feature | `does not include`; `is not shown`; `lacks` |

## Tense Control

| Situation | Tense |
|---|---|
| Old map only | past simple: `was located`, `occupied` |
| Change from old to new | past perfect passive: `had been replaced`, `had been moved` |
| New map state | past simple passive: `was located`, `was used as` |
| Two current layouts | present simple: `contains`, `measures`, `has`, `includes` |

## Band 8 Focus

- Mention both changes and unchanged features.
- Use passive forms because the actor is unknown.
- Use location language: `in the top-left corner`, `on the right-hand side`, `in the central area`.
- Keep the report objective: no opinion like `better`, `more beautiful`, or `more convenient`.
- Aim for about 160 to 190 words.
- For feature comparison, avoid change verbs like `replaced`, `relocated`, or `converted`.
- Compare categories directly: size with size, facilities with facilities.

## Quick Review

Intro = paraphrase.
Overview = biggest changes + stability.
Bodies = compare by area/function, not by year.

Feature comparison = compare layouts by size, capacity, and facilities.
Change over time = compare what changed, what moved, and what stayed.
