# Writing Mistakes

Track repeated writing errors here.

Useful categories:

- Task response
- Coherence and paragraphing
- Weak examples
- Grammar accuracy
- Word choice
- Overgeneralization
- Punctuation

## Task 1 Overview: Comparison Grammar

Pattern to avoid:

`these are [number] categories have...`

Better patterns:

- `[Category A] and [Category B] saw/recorded significant increases, while [Category C] rose more modestly.`
- `There were significant increases in [Category A] and [Category B], compared with a smaller rise in [Category C].`
- `[Category A] and [Category B] experienced the strongest growth, whereas [Category C] showed only a slight increase.`

Rules:

- Do not use `these are` before a full subject + verb.
- Use plural verbs for plural subjects: `Countries 1 and 3 saw`, not `Country 1 and 3 has`.
- Use `in terms of`, not `in term of`.
- Keep comparison clean: `while / whereas / compared with`.

## Speaking Habits That Break Writing

These mistakes happen when the sentence is written like spoken English.

| Spoken Habit | Writing Problem | Better Control |
|---|---|---|
| Starting with `this is / these are` | Creates broken subject + verb structure. | Start with the real subject: `[A] and [B] saw...` |
| Long chain with `and / so / but` | Run-on sentence, weak logic. | Split or use one clear connector: `whereas`, `therefore`, `however`. |
| Missing subject | Sentence fragment. | Every sentence needs a clear subject + verb. |
| Vague `this / these / it` | Reader cannot see the reference. | Name the data/idea again: `this increase`, `these figures`, `this policy`. |
| Direct translation from Vietnamese | Awkward word order or wrong phrase. | Use IELTS-safe sentence patterns. |
| Casual verbs: `get`, `make`, `do` | Meaning becomes too general. | Use precise verbs: `record`, `experience`, `cause`, `lead to`, `result in`. |
| Speaking fillers: `actually`, `basically`, `you know` | Informal and low-value. | Delete them. |
| One idea repeated in many ways | Wordy, no extra meaning. | One clean sentence per idea. |
| Tense switching | Timeline becomes unclear. | Charts/maps: keep tense consistent with the visual. |
| Countable noun errors | Grammar accuracy drops. | Check plural: `figures`, `countries`, `percentages`, `facilities`. |

Quick rule:

After writing a sentence, ask: `What is the subject? What is the verb? What is being compared?`

## Task 1 Data Language Control

| Wrong Pattern | Why It Fails | Better Pattern |
|---|---|---|
| `have the most jump` | informal + wrong collocation | `recorded the sharpest increase` |
| `all 3 country` | spoken wording + plural error | `among the three countries` |
| `the increase is more than double` | the figure doubles, not the increase | `the figure more than doubled` |
| `the buying of goods` | unnatural noun phrase | `goods purchases` / `the figure for goods` |
| `unimaginable / crazy / huge` | subjective or too informal | `dramatic / sharp / substantial` |
| `tiny grow` | wrong word form | `slight increase` / `modest growth` |
| `food consumed` | changes the meaning if chart says bought | `food purchases` / `food bought` |

Quick rule:

Describe data with `figure`, `proportion`, `percentage`, `purchase`, `increase`, `rise`, `fall`; avoid spoken emotional words.
